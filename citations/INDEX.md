## How to add a new citation

1. Create a file in the right subfolder, naming by case/Act/publisher + year (kebab-case).
2. Fill in the frontmatter: `title`, `citation`, `court_or_publisher`, `date`, `primary_url`, `retrieved`, `used_in`, `copyright`.
3. Body:
   - **statutes/cases/advisories/policies/regulators** — full verbatim text where copyright allows (see README.md § Copyright policy).
   - **commentary** — short excerpt + link only, never full reproduction.
4. Add a row to the table above in this INDEX.
5. Reference the file path in the commit message of the site change that depends on it.

## How to refresh a citation

1. Open the file.
2. Re-fetch the `primary_url`.
3. If unchanged, bump the `retrieved` date in the frontmatter and commit.
4. If changed, note the diff in a new changelog section at the bottom of the file (do not delete the old snapshot — version the change).
